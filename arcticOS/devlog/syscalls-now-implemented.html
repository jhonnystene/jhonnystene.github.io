System Calls Now Implemented

I've implemented system calls into arcticOS. It was really the obvious first step towards building a "proper" mobile operating system, and will actually allow me to continue updating phones for longer.

What are system calls?
System calls are a way for applications to "talk" to the operating system - they add lots of modularity and make the development process much easier. Essentially, without system calls, programs would need to either talk directly to the hardware (which *is* permitted on arcticOS, but is not usually the best solution) or they would need to know exactly where the relevant chunks of the OS live. With system calls, a program just needs to say "Hey operating system - I want to do (insert thing here)" and the OS will parse the data it is given to execute the command. How does this improve arcticOS?
The most obvious benefit is it opens the door to potential third-party application support *without* recompiling the OS to add in every app. One less immediately apparent benefit is that it makes the OS much more modular - you'll be able to swap out different pieces and it should all just run. This modularity also brings about what is (in my eyes, anyway) a very important part of arcticOS - I could completely stop supporting one model of phone and spend (close to) zero time working on software for it, but continue to release useful updates for it regularly. The reason I can do this is because arcticOS's modularity means I can "swap out" the core of arcticOS for a newer version and spend very little time getting it to work. That means that even if I completely abandon dumbphones altogether, development I do for a smartphone, computer, etc... will still be somewhat relevant to the dumbphones and will continue to work. This of course doesn't mean I'm going to switch to zero-effort updates right after launch. As long as it is economically viable for me to design & manufacture dumbphones, they'll be getting proper updates. I'll be talking more about my general update plans in a future post.
